# 📖 Project Overview

MedCycle is an AI-powered Smart Medicine Donation and Expiry Tracking System designed to minimize medicine wastage and improve healthcare accessibility. The platform enables users to donate unused medicines, monitors medicine expiry dates, and supports intelligent decision-making through Machine Learning.

> **Note:** This repository showcases the project. The implementation details and source code are intentionally kept private.

---

# 🎯 Problem Statement

Large quantities of medicines are discarded every year after expiry, while many people struggle to access essential medicines.

MedCycle bridges this gap by providing a centralized platform for medicine donation and expiry monitoring.
